# "Pennergame" Webanwendung - Penner Edition

Pennergame ist eine interaktive Fullstack-Webanwendung, die im Kern als moderne Neuinterpretation des Klassikers "Pennergame Sylt" entwickelt wird. Sie kombiniert spielerische RPG-Elemente mit technischem Wissen und einer modernen Systemarchitektur.

## 🚀 Features

### 🏢 Penner-Simulation & Wirtschaft
* **Sammel-System:** Gehe im Stadtpark Flaschen sammeln (10-Minuten-Takt), um dein Startkapital aufzubauen.
* **Finanzmanagement:** Behalte dein Geld im Auge, aber achte auf dein "Becher-Limit" – investiere in Upgrades wie neue Behältnisse, um mehr tragen zu können.
* **Skill-System:** Trainiere deine Fähigkeiten in den Bereichen Angriff (ATT), Verteidigung (DEF) und Geschicklichkeit (DEX), um im Spiel voranzukommen.

### 🏆 Quiz & Trophäen (Legacy-System)
* **Lernquiz:** Teste dein Wissen in technologischen Themen, um zusätzliche Punkte und Trophäen freizuschalten.
* **Dashboard:** Dein zentraler Überblick über Kapital, Inventar, Lernfortschritt und errungene Trophäen.

### 🎮 Minigames
* **Hacker Minigame:** Fordere deine technischen Fertigkeiten in integrierten Minispielen heraus.

## 🛠 Technologien

### Frontend
* **React.js:** Dynamische und responsive Benutzeroberfläche.
* **Axios:** Echtzeit-Synchronisation mit dem Backend für Spiel-Stats.
* **Material-UI:** Integration von modernen UI-Komponenten und Avataren.

### Backend
* **Express.js:** Robuste Server-Infrastruktur.
* **JWT (JSON Web Tokens):** Sichere Authentifizierung und Sitzungsverwaltung.
* **Sequelize ORM:** Effiziente Verwaltung der Datenbank-Modelle und Relationen.

### Datenbank
* **Aiven MySQL Cloud:** Hochverfügbare relationale Cloud-Datenbank für die sichere Speicherung von Benutzerdaten, Spielstatistiken und Quizzes.
